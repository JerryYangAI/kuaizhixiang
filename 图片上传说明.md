## 上传步骤

1. 将图片文件重命名为上述文件名
2. 将图片文件复制到 `public/images/products/` 目录
3. 运行 `git add public/images/products/*.jpg`
4. 运行 `git commit -m "添加产品图片"`
5. 运行 `git push` 推送到 GitHub

## 图片要求

- **格式**: JPG 或 PNG
- **建议尺寸**: 至少 800×800 像素
- **文件大小**: 建议小于 500KB（可压缩优化）
